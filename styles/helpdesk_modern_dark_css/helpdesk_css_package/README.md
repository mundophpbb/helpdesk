# Help Desk Modern Dark CSS

Pacote CSS criado a partir dos templates enviados:
- `helpdesk_my_tickets_body.html`
- `helpdesk_queue_body.html`

## Arquivo principal
- `theme/helpdesk-modern-dark.css`

## Estilo adotado
- Moderno
- Minimalista
- Dark Mode
- Layout responsivo com Grid e Flexbox

## Paleta usada
- Fundo principal: `#0b1220`
- Superfície: `#121a2b`
- Destaque: `#38bdf8`
- Sucesso: `#22c55e`
- Aviso: `#f59e0b`
- Erro: `#ef4444`
- Texto: `#e5edf8`

## Como usar no phpBB
Inclua o CSS no carregamento do tema/extensão, por exemplo no diretório `styles/prosilver/theme/` da sua extensão ou no ponto onde você já registra assets do front-end.

## Observações
O CSS foi pensado para os componentes encontrados nos templates enviados, incluindo:
- cards de resumo
- abas
- filtros
- listas/tabelas de tickets
- badges e pills de status
- painéis colapsáveis (`details/summary`)
- blocos de balanceamento e redistribuição
- estados vazios
- responsividade para tablet e mobile

# MundoPHPBB Help Desk

Version 2.4.0

Generic multilingual help desk extension for phpBB with:
- status, priority, category and department
- assignment and activity timeline
- forum queue, team queue and quick reports
- bulk actions
- operational alerts and SLA indicators
- workflow automation rules

## New in 2.4.0
- ACP settings for workflow automation
- automatic status after staff reply
- automatic status after user reply
- optional auto-assignment on staff reply
- configurable lock on closed / unlock on reopen

## Notes
- after updating, update the extension in phpBB to apply migration `v240`
- purge the phpBB cache after replacing the files
